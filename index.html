构建具有记忆能力的 AI 代理

Letta 是构建有状态 AI 代理的平台。您的代理可以记住过往交互、学习用户偏好，并随时间不断改进。

npm install -g @letta-ai/letta-code
快速开始 → 了解核心概念

为什么选择 Letta？

有状态代理

与传统无状态 LLM 不同，Letta 代理在会话之间保持记忆，能够从经验中学习和改进。

分层记忆系统

代理拥有上下文记忆、记忆块和归档记忆，智能管理上下文窗口限制。

模型无关

支持 Claude、GPT、Gemini 等多种模型，可随时切换，灵活配置。

Letta 产品

Letta Code

在终端中运行的有记忆能力的编程代理。帮助您编写代码、编辑文件、运行程序，并随时间学习您的偏好。
