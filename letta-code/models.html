模型配置

在 Letta Code 中使用任何模型

Letta Code 是模型无关的。代理可以使用 Claude、GPT、Gemini 或其他支持的模型。用户可以随时更改代理的底层模型，甚至在对话过程中。

请访问 leaderboard.letta.com 查看比较模型性能的基准测试结果。

切换模型

在会话中使用 /model 命令切换模型：

> /model

这将打开一个模型选择器，包含两个类别：

  • 已支持 - 预配置的模型，具有优化的设置
  • 所有可用 - 您账户上可用的所有模型，包括自定义 BYOK（自带密钥）模型

使用 Tab 切换类别，方向键导航，j/k 翻页。

您也可以在启动 Letta Code 时指定模型：

letta --model gpt-5-codex
letta -m haiku

工具集

不同的模型被训练为最适合使用不同的工具。例如，GPT-5 模型被训练使用基于补丁的编辑工具，而 Claude 模型更适合使用基于字符串的编辑工具。

Letta Code 通过工具集自动处理这个问题 - 为每个模型系列优化的工具配置。当您使用 /model 切换模型时，Letta Code 会自动切换到最佳工具集。

工具集 优化用于
Default Claude 模型（Anthropic）
Codex GPT 模型（OpenAI）
Gemini Google 模型

手动覆盖工具集

如果您想强制使用特定工具集：

letta --toolset codex

或在交互模式中：

> /toolset

推荐模型

🥇 最佳性能

  • Claude Opus 4.5 - 最高质量输出
  • GPT-5.2-Codex - OpenAI 最佳编程模型

⚡ 性价比之选

  • Claude Sonnet 4.5 - 速度与质量的平衡
  • GPT-4o - 快速响应
  • Claude Haiku - 最快最便宜

Google 模型

gemini-1.5-pro      # Gemini 1.5 Pro

自带密钥 (BYOK)

您可以使用自己的 API 密钥来直接向提供商付费。使用 /connect 命令配置：

> /connect

支持的提供商：

  • Anthropic (Claude)
  • OpenAI (GPT)
  • Google (Gemini)
  • zAI
  • Minimax
  • OpenRouter
  • Azure OpenAI
  • Amazon Bedrock (Claude)
  • Together AI

更多详情请参阅 提供商 页面。
