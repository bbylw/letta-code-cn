提供商

Letta Code 支持的模型提供商和计划

Letta API

Letta API 支持广泛的模型提供商，包括 OpenAI、Anthropic、Google Gemini、zAI 等（查看可用模型的完整列表）。

Letta Pro Plan 或 Letta Max Plan 的用户可以通过订阅直接访问 Letta API 上的任何模型，并通过按需付费额度支付额外使用费用（查看使用页面 配置自动充值）。

BYOK 认证

使用 BYOK 认证时，您直接由 LLM API 提供商以其自己的费率计费。

您也可以通过 /connect 命令添加自己的 LLM API 提供商密钥。这允许您使用自己的 API 密钥访问以下提供商：

  • OpenAI
  • Anthropic
  • Google Gemini
  • Z.ai
  • Minimax
  • OpenRouter
  • Claude on Amazon Bedrock
  • Azure OpenAI
  • Together AI

您也可以将 Letta Code 与现有的 OpenAI ChatGPT Pro/Plus 计划（包括 Codex 使用）、zAI GLM 和 Minimax 编程计划一起使用。

Codex (ChatGPT Pro / Plus)

将您的 ChatGPT Pro / Plus 订阅连接到 Letta Code，使用您现有的计划访问 OpenAI 模型。

认证

使用 /connect 命令通过 OpenAI 认证：

> /connect codex

这将打开一个浏览器窗口，授权 Letta Code 访问您的 OpenAI 账户。连接后，您的 ChatGPT Pro / Plus 计划额度将用于 OpenAI 模型，提供商名称为 chatgpt-pro-plus。

要断开与 OpenAI 账户的连接，您可以运行：

> /disconnect codex

OpenAI

使用您自己的 OpenAI API 密钥：

> /connect

选择 OpenAI 并输入您的 API 密钥。

Anthropic

使用您自己的 Anthropic API 密钥访问 Claude 模型：

> /connect

选择 Anthropic 并输入您的 API 密钥。

Google Gemini

使用您自己的 Google AI API 密钥访问 Gemini 模型：

> /connect

选择 Google Gemini 并输入您的 API 密钥。

Z.ai

使用 Z.ai API 访问 GLM 模型：

> /connect

选择 zAI 并输入您的 API 密钥。

zAI GLM 编程计划

要连接您的 zAI GLM 编程计划，只需运行 /connect，然后选择 zAI 连接您的 zAI API 密钥。

Minimax

使用 Minimax API：

> /connect

选择 Minimax 并输入您的 API 密钥。

Minimax 编程计划

要连接您的 Minimax 编程计划，只需运行 /connect，然后选择 Minimax 连接您的 Minimax API 密钥。

OpenRouter

通过 OpenRouter 访问多种模型：

> /connect

选择 OpenRouter 并输入您的 API 密钥。

Claude on Amazon Bedrock

通过 Amazon Bedrock 使用 Claude 模型：

> /connect

选择 Bedrock 并配置您的 AWS 凭证。

Azure OpenAI

使用 Azure OpenAI 服务：

> /connect

选择 Azure OpenAI 并配置您的端点和密钥。

Together AI

使用 Together AI 访问开源模型：

> /connect

选择 Together AI 并输入您的 API 密钥。

断开连接

要断开任何提供商：

> /disconnect <provider-name>

故障排除

API 密钥问题

如果您的 API 密钥不起作用：

  • 确保密钥有效且未过期
  • 检查密钥是否有足够的额度
  • 验证密钥是否有访问所需模型的权限

连接问题

如果连接失败：

  • 检查您的网络连接
  • 验证您是否可以使用相同的密钥直接访问提供商的 API
  • 尝试运行 /disconnect 然后重新连接
